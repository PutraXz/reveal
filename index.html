Backend

Backend merupakan bagian dibalik layar dari sebuah website, semua logic dari website diatur oleh backend seperti login,register,crud dsb. bahasa pemograman umum yang sering backend gunakan untuk membangun website diantara nya sebagai berikut .

PhP

Php atau disebut juga dengan hypertext preprocessor merupakan bahasa pemograman yang digunakan secara luas untuk pembuatan atau pengembangan pada sebuah situs web, bahasa pemograman ini sudah meningkat jauh pesat dan bahasa ini sekarang sudah memiliki framework diantara lainnya adalah sebagai berikut:

Framework

Sesuai dengan namanya sendiri, Framework merupakan kerangka kerja untuk mengembangkan aplikasi berbasis website atau dekstop, disini framework memiliki fungsi yang sangat membantu developer dalam pembuatan atau pengembangan aplikasi.

Laravel

Laravel merupakan framework dari bahasa pemograman php yang populer dan banyak digunakan diseluruh dunia, framework ini populer dikarenakan fitur nya yang sangat membantu developer diantara lainnya adalah Authentication, Routing, Eloquent
